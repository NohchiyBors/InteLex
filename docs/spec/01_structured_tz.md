# InterLex: Структурированное ТЗ на разработку сайта

## 1. Общая информация

### 1.1 Проект
- Название: `interlex.kz`
- Формат: корпоративный B2B-сайт юридической и бизнес-консалтинговой компании
- Основной тип сайта: полноценный корпоративный сайт + каталог услуг + лидогенерация + контент-платформа
- Языки: `RU`, `EN`, `ZH`, `KK`, `KA`
- Доменная стратегия:
  - `interlex.kz` — основной язык `RU`
  - `interlex.ge` — основной язык `EN`
  - `ZH`, `KK`, `KA` — дополнительные языковые версии

### 1.2 О компании
InterLex оказывает услуги по регистрации и сопровождению бизнеса для иностранных клиентов в двух юрисдикциях: Казахстан и Грузия. Основное позиционирование: один подрядчик закрывает полный цикл юридических, регуляторных, бухгалтерских и инвестиционных задач.

### 1.3 Основное УТП
- Две юрисдикции в одном контракте: Казахстан + Грузия
- Полный цикл услуг: от регистрации до операционного сопровождения
- Работа с иностранными инвесторами, холдингами, IT-компаниями и M&A-сделками
- Экспертиза по СЭЗ/СИЗ, AIFC, Astana Hub, Virtual Zone, Due Diligence, GR
