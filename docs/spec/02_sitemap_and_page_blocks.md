- Финальный список разделов этапа 1
- Наличие отдельных страниц под каждую СЭЗ
- Объём китайской версии на запуск
- Нужен ли полноценный личный кабинет: по текущему брифу нет
- Нужна ли оплата на старте: скорее второй этап
- Какие интеграции обязательны в этапе 1, а какие можно отложить
